# String Art Stepper
A graphical asistent for follow the steps provides by the [String art generator](https://www.stringartgenerator.app) app.
Provide a friendly UI to create beautiful string art.

## How to use
1. First you have to generate the JSON file with all the necesary data, with [String art generator](https://www.stringartgenerator.app).![String art generator capture](./images/step_1.png)
2. Then, upload the JSON file in the application to parse de data and prepare the assistant.![Screen capture uploading a file](./images/step_2.png)
3. Start following the steps with the buttons or the arrow keys!![Step viewer](./images/step_1.png)

## Libraries used 
 - React for UI
 - FontAwesome for React
 - Vitest to testing
